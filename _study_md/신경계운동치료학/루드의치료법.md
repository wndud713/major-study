# Rood's Theory — 루드 치료법

## 루드 소개


### 인물

- **Margaret S. Rood (1909~1994)** — 물리치료사 + 작업치료사

- **학력:** 1932년 작업치료과 학사, 물리치료과 석사학위 취득

- **경력:** 1933~1936 위스콘신 정신병원 작업치료사, 1937~1943 뇌성마비 클리닉 실장



### 치료가설

- **핵심 가설:** 적절한 감각 자극이 특정 운동반응을 이끌어내고, 운동반응은 정상발달순서로 나타나며, 심리·신체·자율신경 요소들과 상호적 관련성을 지님

- **운동기억흔적(motor engrams):** 자극에 의해 생기는 지워지지 않는 기억 — 출생 시 원시반사패턴 → 감각자극으로 수정·사용 → 고위중추(겉질) 성숙으로 원시패턴 소멸

- **치료 적용:** 조절된 감각자극 + 목적있는 반응의 요구(개체발생학단계 + 활동) → 치료에 적용



### 영향을 준 인물·후속 이론

- **관련 인물:** Joy Huss, A. Jean Ayres(감각통합치료), Shirley StockMeyer

- **감각통합치료(Ayres, 1920~1988):** 뇌가 신체·환경 감각을 조직화하여 신체를 효과적으로 사용하는 신경생리학적 과정 — 루드 치료의 이론적 근간

## 루드접근법의 주요 구성요소 (4원칙)


### 4원칙 요약


- ① 원하는 근 반응과 긴장 정상화를 위해 적절한 자극 이용

- ② 감각운동조절은 신경발달학적 이론에 근거하여 단계별 실시

- ③ 원하는 동작패턴 유발을 위해 목적있는 동작 이용

- ④ 감각운동의 반복을 통해 운동학습 성취




### 4원칙 상세

- **①:** 정확한 조절된 감각입력 → 특히 운동조절 획득 초기단계에 효과적

- **②:** 현재 발달수준에서 시작 → 점차 높은 감각운동조절 수준으로 (머리→발, 몸쪽→먼쪽, 굽힘근→폄근, 내재근→외재근)

- **③:** 겉질밑 중추 작용에서 유발되는 동작패턴(몸중심, 근위부) — 겉질아래중추는 복잡한 운동패턴 조절·교정, 대뇌겉질은 행위의 목적에 집중

- **④:** 단기기억 → 반복 → 장기기억



### 치료원칙

- **긴장성 목반사·미로반사:** 감각 자극 효과를 증진·저지 가능 — 목 굽힘→팔 굽힘·다리 폄 / 목 폄→팔 폄·다리 굽힘 / 목 돌림→팔·다리 폄 긴장도 증가

- **특정수용기 자극의 3가지 반응:** ①자율신경계 항상성반응, ②척수·뇌줄기 반사적 보호반응, ③적응된 반응(신경계 모든 부위 통합)

- **근육 역할:** 근육은 각기 다른 역할 → 힘든활동근육이 가벼운활동근육보다 먼저 통합되어야 함



> 💡 💡 **핵심:** 감각자극으로 근육의 운동반응을 유발 → 목적있는 활동 가능케 함. 수의적 움직임은 고위중추 통제, 겉질아래 수준도 과제수행에 기여

## 계통발생학적 운동발달패턴


### 발달순서

- **발달 원칙:** 굽힘→폄 / 모음→벌림 / 자뼈패턴(ulnar)→노뼈패턴(radial) / →돌림



### 운동조절 4단계


1단계 운동성항중력적 움직임 발달, 상반억제단계 — 초기 운동성
2단계 안정성협력수축 단계 — 항중력 체중지지자세 유지 위해 작용근·대항근 동시수축
3단계 조절된운동성안정성 기초 — 먼쪽부위 고정+몸통 움직임 (예: 네발기기에서 팔다리 체중지지+몸통 다방향 움직임)
4단계 숙련(기술)몸통 안정+먼쪽 자유 움직임 (예: 팔꿈치지지 엎드리에서 한쪽팔 자유 사용)

- **전체 흐름:** 집단운동패턴 → 자세안정성 획득 → 먼쪽 신체지지(체중이동) → 정상패턴 자유 동작



### 근골격계 운동발달 9단계


| 단계 | 자세 | 운동조절단계 | 설명 |
| --- | --- | --- | --- |
| 1 | 움추린 바로누운자세 | 운동성 | T10 부위 전체적 굽힘반응, 몸쪽·몸통 힘 요구 |
| 2 | 돌아눕기 | 운동성 | 외측몸통 활성화, 안뜰기관·목·눈 근육 자극 |
| 3 | 배를 축으로 엎드리 | 운동성 | 엎드려 전체 폄, 대칭성긴장성목반사·미로반사 통합 |
| 4 | 목근육 협력수축 | 안정성→조절된운동성 | 중력 대항 머리 들기 — 굽힘근·폄근 협력수축 |
| 5 | 팔꿈치지지 엎드리 | 안정성→조절된운동성→숙련 | 어깨뼈 안정성, 대칭성긴장성목반사 억제, 고개돌려 환경탐색 |
| 6 | 네발기기자세 | 안정성→조절된운동성→숙련 | 목·어깨 안정성, 아래몸통·다리 협력수축, 점차 체중이동 |
| 7 | 서기 | 안정성→조절된운동성→숙련 | 정적서기→팔 움직임 허용, 바로세우기·평형반응 출현 |
| 8 | 걷기 | 숙련 | 아래몸통·다리 협력수축 + 체중이동 |



### 전형적 Rood vs 재구성된 Rood


| 전형적 Rood(계층이론) | 재구성된 Rood(운동조절이론) |
| --- | --- |
| 근긴장도 정상화 = 움직임 전제조건 | 근긴장도와 운동조절은 서로 협응 효과 |
| 치료는 기능 발달수준에서 시작 | 굴곡·신전 패턴은 협응 효과 |
| 근육반응 재교육은 반복으로 | 근육반응 반복 → 움직임 패턴 형성 |
| 움직임은 기능목표를 향해 지도 | 목적/목표 방향은 움직임과 상호작용 |
| 실생활 근접 활동 → 효율성·일반화↑ | 실생활 근접 활동 → 효율성·일반화↑ |

## 가벼운 활동 근육 (Light Work Muscle)


### 특성

- **유형:** 위상성 근육(Phasic flexors), Type II 섬유

- **신경 경로:** 가쪽겉질척수로, 적색척수로와 연결

- **기능:** 수의적 또는 의지적 움직임 / 굽힘근, 모음근

- **활성화:** 낮은 문턱값 외수용기 자극에 의해 활성화

- **억제:** 비저항수축에 의해 억제

---

## 힘든 활동 근육 (Heavy Work Muscle)


### 특성

- **유형:** 긴장성 근육(Tonic extensor), Type I 섬유

- **신경 경로:** 안뜰척수로, 그물척수로, 바깥피라미드(추체외로)와 연결

- **기능:** 반사적인 움직임 / 폄근, 벌림근

- **활성화:** 높은 문턱값 수용기 자극에 활성화 / 강한 저항, 지속적 뻗침에 의해 활성화



### 전체패턴

- **패턴1 (몸쪽패턴, 위상굽힘근패턴):** 입 상부 중앙 가볍게 누르기 → 손목·손가락 폄 + 팔꿈치·어깨 굽힘 / 다리 굽힘(아동), 성인은 배꼽 자극 시 다리 굽힘

- **패턴2 (몸바깥쪽패턴, 긴장성폄근패턴):** 근육 내 신장감수기에 반복적 빠른 솔질 + 미로자극 → 뻗침성 폄근 패턴 촉진 / 진동, 압력 등에서도 활성화



> 💡 💡 치료 순서: 힘든활동근육(안정화) 먼저 통합 → 가벼운활동근육(이동) 나중

## 촉진법 — 운동성 촉진(위상반응)


### 촉진법 원리

- **원리:** 운동신경원(motor neuron) 이음부(연접부, synapse)의 문턱값(threshold)을 낮추는 조치


| 감각 | 자극 |
| --- | --- |
| 피부감각 | 얼음, 빠른솔질, 가벼운 접촉 |
| 고유수용감각 | 관절압박, 빠른 뻗침, 두드리기 |
| 교감신경계 | 빠르고 거칠게 쓰다듬기 |



### 1) 가벼운 접촉 (Light Touch)

- **수용기:** 털 종말기관 + 자유신경 종말 수용기 활성화 → 운동근 수축 촉진

- **방법:** 손가락 끝, 털, 면봉으로 가벼운 접촉/쓰다듬기 — 초당 2회씩 10회, 3~5세트 반복

- **적용:** 손바닥·발바닥·손가락·발가락 web 배측을 가볍게 문지르면 → 위상성 도피동작(phasic withdrawal motion) 유도



### 2) 얼음자극 (Fast Ice)

- **원리:** 저긴장성 이완 상태 표재근에 적용 → A-δ섬유 활성화 → 회피반응

- **특수 적용:** 이완성 마비에서만 사용 / T7-T9 피부분절, 복부 상 우측 1/4에 A섬유 얼음자극 → 횡격막 수축 촉진

- **금기:** ①삼차신경 지배 목 위·귓바퀴 적용 금지, ②자유신경종말 최대결집부위(몸 중앙축) 금지, ③C4-5 척수손상 환자(자율반사이상), ④심혈관계 과거력 환자(협심증·부정맥 유발)



### 3) 빠른 뻗침 (Quick Stretch)

- **원리:** 고유수용기 활성화 — 근육방추 흥분 → 위상뻗침반사(phasic stretch reflex) 유발



### 4) 가볍게 두드리기 (Tapping)

- **방법:** 힘줄 또는 근육 힘살 위를 손끝으로 가볍게 두드리기 → 근육방추 자극 → 근 긴장도 증진

## 촉진법 — 안정성 촉진(긴장반응)


### 1) 강한 관절압박 (Heavy Joint Compression)

- **목적:** 관절주위근의 협력수축 촉진 — 주로 긴뼈의 장축에 적용



### 2) 뻗침압력 (Stretch Pressure)

- **방법:** 표재근육 위 피부에 엄지·검지·중지를 놓고 손가락이 멀어지도록 강한 압력과 뻗침 적용 — 자극은 3초 이내



### 3) 빠른솔질 (Fast Brushing)

- **원리:** C섬유 자극 — 피부분절 위에 3~5초간 적용 → 30초 후 다시 반복, 주로 등쪽 근육

- **금기:** 위쪽목손상·뇌줄기손상 환자(자율반사이상·무의식), S2-4 자극 시 방광정체 유발, 귓바퀴솔질은 미주신경자극하여 심호흡에 영향



### 4) 얼음자극 (Slow Ice)

- **방법:** 3~5초 문지른 후 얼음물 닦아내기 → C섬유 자극하는 고역치 자극법 → 긴장반응 촉진



### 5) 뼈압박 (Bone Pressure)

- **예시:** 발뒤꿈치 가쪽면 위에 압력 적용 → 안쪽 등쪽굽힘근 촉진, 장딴지근 억제



### 6) 저항 (Resistance)

- **안정성:** 근육이 짧아진 상태에서 최대저항 적용 → 안정성 촉진

- **운동성:** 적당한 저항 제공 → 운동성 촉진

## 억제 기법


### 억제 목적

- **대상:** 근육이 과도하게 긴장되어 있거나 강직이 있는 경우



### 1) 부드럽게 흔들기 (Slow Rocking)

- **방법:** 바로누운자세에서 옆으로 천천히 구르게 하기 / 흔들의자에서 몸을 천천히 흔들기

- **주의:** 너무 빠른 흔들기는 안뜰기관 자극 → 오히려 촉진됨



### 2) 천천히 쓰다듬기 (Slow Stroking)

- **방법:** 척추뒤가지 부위를 가볍게 압력 주면서 천천히 쓰다듬기 — 뒤통수→꼬리뼈 방향

- **지속:** 교대로 손 바꿔가며 이완될 때까지 3~5분 지속



### 3) 약한 관절압박 (Light Joint Compression)

- **방법:** 강직 있는 관절에 뼈의 긴축에 인체 무게보다 적은 무게로 가벼운 압박



### 4) 지속적 뻗침 (Prolonged Stretch)

- **방법:** 몇 분~수 주일 동안 근육을 길어진 위치에 유지 → 근방추 적응



### 5) 천천히 굴리기 (Slow Rolling)

- **방법:** 옆으로 누운 자세에서 어깨·가슴·골반 가쪽에 손을 놓고 엎드린자세↔바로누운자세로 율동적 굴리기

- **방향:** 비마비측 → 마비측



### 기타 억제 기법

- **힘줄 부착부 압박:** 근육의 힘줄이 뼈에 부착하는 곳(기시부, 정지부)에 압박 적용

- **장시간 신장 유지:** 오랜 시간 동안 정상적인 길이보다 길어진 위치에 근육이 놓이도록 유지

- **역방향 수축:** 근육의 길이가 늘어나는 방향으로 짧고 강력한 수축 유도



### 운동성 촉진법 요약

- **가벼운 촉진(light touch):** 운동근(mobilizing muscle) 수축 촉진

- **얼음조각 빠르게 문지르기(quick swipes):** 낮은 역치 A섬유(A-fiber) 자극

- **빠르고 가볍게 근육 신장:** 근방추 흥분 → 위상뻗침반사(phasic stretch reflex) 유발

- **타진(tapping):** 힘줄·근육힘살에 대한 두드리기 → 근수축 촉진



### 안정성 촉진법 요약

- **빠른솔질(fast brush)**

- **얼음조각 천천히 문지른 후 닦아내기**

- **근육힘살(muscle belly)에 지속적 압력**

- **내재근(intrinsic muscle)에 대한 신장**

- **체중지지 자세(weight bearing)**

- **저항(resistance)**

- **강한 관절압박(heavy joint compression)**



### 하스(Joy Huss)의 치료

- **개요:** 루드의 접근법을 토대로 긴장저하와 긴장항진에 대한 치료지침을 만들어 적용



> 💡 💡 억제 후 → 촉진 기법 적용 → 정상 운동 패턴 유도

## 8단계 발달 순서



1단계배와기 (Supine withdrawal) — 굴곡 패턴
2단계구르기 (Rolling) — 몸통 회전
3단계복와위 (Prone on elbows) — 팔꿈치 지지
4단계네발기기 (All-fours/Quadruped)
5단계무릎서기 (Standing on knees)
6단계반무릎서기 (Half-kneeling)
7단계서기 (Standing)
8단계보행 (Walking)



> 💡 💡 치료는 환자의 현재 발달 수준에서 시작 → 순차적으로 상위 단계 진행

## 생명기능 발달을 위한 감각자극


### 생명기능 운동발달 8단계


| 단계 | 기능 | 운동조절단계 |
| --- | --- | --- |
| 1 | 들숨 | 운동성 |
| 2 | 날숨 | 운동성 |
| 3 | 빨기 | 안정성(혀·입·목 주변근육 협력수축) |
| 4 | 액체삼키기 | 조절된 운동성 |
| 5 | 발성 | 운동성(혀 뒤), 숙련(혀 앞) |
| 6 | 씹기 | 조절된 운동성(중력 대항 씹기근육) |
| 7 | 고체삼키기 | 조절된 운동성 |
| 8 | 말하기 | 기술 단계 |



### 혀 움직임 감각자극

- **운동성 촉진:** 혀 끝에 압력 적용 → 혀 내밀기 촉진 / 앞쪽 치아 뒤 입천장을 가볍게 문지르기 → 혀 거상 촉진

- **혀 뒤당김 촉진:** 혀 뒤당김근에 압력 제공 / 혀 아래 부위 수동적 진동

- **안정성 촉진:** 빨대를 빨게 하여 혀·목 주위 근육의 안정성 증진

- **조절된 운동성:** 설압자로 혀 윗부분 3~5초 지속적 누르기 → 처음 구토·물기반응 → 내성 → 혀 후방 올림(elevation) + 삼키기 패턴 발달

- **씹기훈련(조절된 안정성):** 입둘레근, 목뿔근, 가쪽날개근, 목뿔위근, 두힘살근 등에 얼음자극·뻗침자극



### 특수감각 자극

- **목적:** 뇌신경 촉진 + 자율신경계 자극 — 후각·미각·청각·시각 자극을 근육 촉진·억제에 사용

- **청각:** 큰소리가 자극적(촉진 효과)

- **시각:** 단조롭고 옅은 색 환경 → 이완·수면 촉진 / 원색적·자극적 환경 → 촉진 효과
